nternal class Program
{

        {
            var arithmoi = new List<int>() { 1, 4, 3, 1, 53, 2, 53 };

			foreach( var item in arithmoi)

Ορίζει μια λίστα ακεραίων (integers) με το όνομα arithmoi και αρχικοποιεί τη λίστα με κάποιες τιμές.

Προσθέτει έναν νέο αριθμό στη λίστα arithmoi χρησιμοποιώντας τη μέθοδο Add(34).

Εκτελεί έναν βρόγχο foreach που διασχίζει όλα τα στοιχεία της λίστας arithmoi. Για κάθε στοιχείο στη λίστα, εκτυπώνει την τιμή του στοιχείου στην κονσόλα.

Η χρήση της λέξης κλειδί var σημαίνει ότι ο τύπος της μεταβλητής item προσδιορίζεται αυτόματα από τον τύπο των στοιχείων που περιέχονται στη λίστα, 
ο οποίος στην περίπτωση αυτή είναι int.
